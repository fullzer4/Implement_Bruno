<h2>Esse projeto esta usando Nextjs, mongoDB, Typescript</h2>
<h2>Como configurar: </h2>
<h3>Primeiro criar o arquivo env.local seguindo o padrao abaixo: </h3>

```bash
MONGODB_URI= uri mongoDBServer
JWT_SECRET= secret
```

<h3>Depois so rodar: </h3>

```bash
yarn dev
```
